# Установка сборки через приложение [CurseForge](https://www.curseforge.com/minecraft/search?page=1)
#### Для данных способов нужна лицензия, если у вас оной нет, вам [сюда](https://github.com/BMProjects-Development/FAQ/blob/main/RU/%D0%A3%D1%81%D1%82%D0%B0%D0%BD%D0%BE%D0%B2%D0%BA%D0%B0/installation-tlauncher.md)
### Способ 1. Установка сборки скачиванием через приложение (основной)

1. Качаем и устанавливаем приложение [CurseForge](https://www.curseforge.com/download/app) (standalone версия желательно, если вам не нужен Overwolf).
2. [Регистрируемся](https://sso.curseforge.com/oidc/interaction/HCg-KuLnUUzavLY9sP1kd/init-registration) или логинимся через представленные способы.
3. Справа сверху нажимаем Log in, логинимся в аккаунт Minecraft.
4. В поиске пишем имя нужной сборки или нажимаем Discover\Browse и ищем там любую которая вам нравится.
5. Нажимаем скачать, ждём, нажимаем играть.

### Способ 2. Установка сборки через архив
1. В своих модпаках нажимаем создать.
2. Указываем имя будущей сборки, версию, модлоадер и версию модлоадера. 
( например для ftb stoneblock 4, нам нужно будет - 1.21.1, neoforge 21.1.213 )
3. Создаем, слева от кнопки play есть три точки, нажимаем туда, выбираем open folder.
4. Кидаем в открывшуюся папку файлы нужной вам сборки.
5. Играем.

### Способ 3. Импорт заранеее экспортированой/скачаного архива сборки 
#### Данный способ работает только для сборок которые есть на сайте CurseForge ( допустим наш перевод stoneblock4 )
1. Заходите на нужную вам сборку
2. Нажимаете скачать, качается маленький зип архив
3. Кидаем этот архив в окно CurseForge или нажимаем импорт ( лучше нажимать импорт)
4. Ждём пока скачается
5. Играем
