# Установка модов Hytale

## ❗️Способ первый. Установка через официальный лаунчер

Для установки нашей сборки Вам необходимо:
1. Скачать и установить саму игру [Hytale](https://hytale.com/).
2. Скачать саму сборку - [Better Hytale](https://github.com/BMProjects-Development/Better-Hytale).
3. Открыть папку с игрой Hytale/UserData/Mods.
4. Перенести все моды из сборки в эту папку.
5. Зайти в игру и при создании мира выбрать все моды из списка.
6. Готово!
